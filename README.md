# Hi, I'm Anis 👋

Master's student in **Network Administration & Security** at Université Abderrahmane Mira Béjaïa, Algeria — but my interests don't stop at the network layer.

I'm passionate about pretty much everything computer science has to offer: from low-level networking and troubleshooting to system design and architecture, from full-stack development and APIs to DevOps practices and deployment pipelines. I enjoy understanding how things work end-to-end, and I like building projects that span multiple layers of the stack rather than staying in one lane.

Whether it's configuring a switch, designing a distributed system, shipping a web app, or setting up a CI/CD pipeline — I'm interested.
